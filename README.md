# Travel Destination Recommendation System (Backend)

## Description
Backend for a MERN-based travel recommendation system that suggests destinations based on user preferences.

## Tech Stack
- Node.js
- Express.js
- MongoDB

##  How To  Run
Open a terminal and navigate to the backend folder:
cd travel_backend

Install backend dependencies:
npm install


Create a .env file inside the travel_backend folder and add:
MONGO_URL=your_mongodb_connection_string
PORT=5000

Start the backend server:
node server.js

## API Endpoint
POST /api/recommend
